看书时候遇到这样一个例子，依次点击4个li标签，正确的运行结果是？

本来天真的以为，3 3 3 3 嘛，每个函数的作用域链中都保存着return 的活动对象，变量i的值都应该是3
but，答案是4 4 4 4 ，诶诶诶？
后来想了想，for循环可以等价于下面的while循环

while(i<4){
console i;
i++;
}

这货的循环停止后的i是4啊~
